# FahriXZ Store

Website marketplace jasa digital modern dengan:

- Firebase Authentication
- Realtime Database
- Wallet System
- Withdraw System
- Mission Reward
- Admin Dashboard
- Tracking Order
- Invoice System
- PWA Support
- Upload File
- Notification System
- Glassmorphism UI

## Fitur

### User
- Login/Register
- Google Login
- Wallet
- Order Tracking
- Voucher
- Membership

### Admin
- Kelola Order
- Kelola Produk
- Statistik
- Broadcast Promo

### Teknologi
- HTML5
- CSS3
- JavaScript ES6
- Firebase
- PWA

## Cara Install

1. Upload ke hosting
2. Hubungkan Firebase
3. Isi firebaseConfig
4. Jalankan website
